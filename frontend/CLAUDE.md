# Frontend Development Guide

## Quick Start
```bash
npm install          # Install dependencies
npm run dev          # Start Vite dev server (port 5173)
npm test             # Run tests
npm run lint         # Lint
npx tsc --noEmit     # Type check
npm run build        # Production build
```
